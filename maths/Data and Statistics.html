Data and Statistics

1. Introduction to Data

Definition: Data refers to a collection of facts, such as numbers, measurements, or observations, used for analysis.
Types of Data:
- Qualitative Data: Descriptive data that can be categorized (e.g., colors, names).
- Quantitative Data: Numerical data that can be measured (e.g., height, weight).

2. Collecting Data

Methods of Collection:
- Surveys and Questionnaires: Gathering information from a group of people.
- Experiments: Conducting experiments and recording results.
- Observations: Noting down observations from natural occurrences.

3. Organizing Data

Data Tables: Arranging data in rows and columns for easy reading.
Frequency Tables: Showing how often each value or category occurs.

4. Representing Data

- Bar Graphs: Used to represent categorical data with rectangular bars.
- Histograms: Similar to bar graphs but used for continuous data divided into intervals (bins).
- Pie Charts: Represent data as slices of a circle, showing the proportion of each category.
- Line Graphs: Show data points connected by lines, ideal for tracking changes over time.

5. Measures of Central Tendency

Mean: The average of a data set.
Formula: Mean = (Sum of all values) / (Number of values)
Example: For data set {2, 4, 6}, Mean = (2 + 4 + 6) / 3 = 4

Median: The middle value when data is arranged in ascending or descending order.
Example: For data set {1, 3, 7}, Median = 3

Mode: The most frequently occurring value in a data set.
Example: For data set {2, 3, 3, 4, 5}, Mode = 3

6. Probability Basics

Definition: The likelihood of an event occurring.
Formula: Probability = (Number of favorable outcomes) / (Total number of outcomes)
Example: Tossing a coin (Probability of heads = 0.5)
