Linear Equations in Two Variables

1. Introduction

A linear equation in two variables is an equation that represents a straight line when plotted on a graph. It has the general form: ax + by = c where a, b, and c are constants, and x and y are the variables.

2. Graphical Method

To solve a system of linear equations graphically, follow these steps:

  1. Write each equation in slope-intercept form: y = mx + b, where m is the slope and b is the y-intercept.
  2. Plot the lines: On a graph, plot each line represented by the equations.
  3. Find the point of intersection: The point where the lines intersect is the solution to the system of equations.

Example: Solve the system of equations graphically:
2x + 3y = 6
x - y = 1

3. Substitution Method

In the substitution method, you solve one of the equations for one variable and then substitute that expression into the other equation.

  1. Isolate one variable: Solve one equation for one variable in terms of the other variable.
  2. Substitute: Substitute the expression into the other equation.
  3. Solve: Solve the resulting equation for the remaining variable.
  4. Back-substitute: Substitute the value found back into one of the original equations to find the value of the other variable.

Example: Solve the system using substitution:
y = 2x + 3
3x - y = 7

4. Elimination Method

The elimination method involves adding or subtracting the equations to eliminate one of the variables.

  1. Align the equations: Write the equations in standard form.
  2. Eliminate one variable: Add or subtract the equations to eliminate one of the variables.
  3. Solve for the remaining variable: Solve the resulting equation.
  4. Back-substitute: Substitute the value found back into one of the original equations to find the value of the other variable.

Example: Solve the system using elimination:
3x + 4y = 10
2x - y = 3

5. Applications

Linear equations in two variables are used in various real-life applications, such as:

  • Budgeting: To find how much to allocate to different categories while staying within a budget.
  • Mixture Problems: To determine the quantity of different substances to mix to achieve a desired result.
  • Optimization: To optimize resource allocation for projects or tasks.
